# websafe-font
font-family / font-size / line-height / color / background-color / letter-spacing of reading friendly websites   
本markdown文档记录了一系列「阅读友好」网页的网络安全字体相关 css 属性  
*测试分辨率1920*1200*

## why do i need it
相信我，中文排版和英文排版是两个概念。  
既然不懂，那么就从中文网页中寻找规律。
